# Title – Pochita Jump!

## Repository
https://github.com/ConnerLongoria/LongoriaConner_PFDA_Final

## Description
Help Pochita eat as much tasty bread without falling down, and maybe even see some familiar friends along the way. Pochita Jump is a jumping platformer showcasing the connection between art in games & the code that brings it all to life!

## Features
- Feature 1: Jumping Pochita
	- The player will be able to control Pochita like any normal platformer, but specifically in Pochita jump, the player can only jump by using platforms as boosts. For an example of this, see the game “Doodle Jump”. However, unlike said game, Pochita Jump will not have different platforms, instead the distance between them will grow to make it more challenging (but not impossible).
- Feature 2: Bread Collection
	- On some of the platforms there is a chance for bread to appear. If collected by the player controlled Pochita, then the player’s points will go up accordingly. Furthermore, there is an even smaller chance the bread has jam, in which the player’s points will increase slightly more. If the player misses a platform and falls the game will end and display the total points collected in a round.
- Feature 3:  Easter Egg Random Events
	- As this game utilizes the character “Pochita” from “Chainsaw Man” I wanted to include references along the way in the background of the game. To do this, every time bread is collected there will be a slight chance of other well-known characters appearing. To showcase this the chances will be adjusted, but in the normal version it would be much lower.

## Challenges
- Coding the physics of Pochita’s falling & jumping.
- Creating the art for all assets used within the game.
- Implementing said assets into the game without issues.
- Consistently saving & committing code written for the game.

## Outcomes
Ideal Outcome:
- The ideal outcome for this game is to make a 1:1 recreation of “Doodle Jump”. This also includes the complexity of the point system that is planned.
Minimal Viable Outcome:
- A basic jumping platformer without very complex sprites or game mechanics. Just a functional game that is still enjoyable to the player.
